Current Smoothing Algorithms

timeline Linear Interpolation

Creates intermediate points between start and end positions to smooth mouse movement.
  • Uses easing function (easeInQuad)
  • Fixed 10 interpolation steps
  • Emits both X and Y events
lerp(start, end, t) = start × (1 − t) + end × t

filter_list Low-pass Filter

Attenuates high-frequency signals (rapid changes) while allowing low-frequency signals to pass.
  • Fixed alpha value (0.9)
  • Stores previous smoothed values
  • Filters mouse movement events
smoothed = α × newInput + (1 − α) × previousSmoothed

analytics Kalman Filter

Predicts mouse position using previous state and updates with new measurements.
  • Fixed process noise (0.1)
  • Fixed measurement noise (0.5)
  • Two-step process: predict and update
Prediction: x̂(k|k-1) = F·x̂(k-1|k-1)
Update: x̂(k|k) = x̂(k|k-1) + K·[z(k) - H·x̂(k|k-1)]
