Other Smoothing Algorithms to Consider

show_chart Moving Average

Averages recent data points to reduce noise
check_circle Simple to implement
check_circle Window size adjustable
check_circle Weighted version available

timeline Exponential Moving Average

Gives more weight to recent data points
EMA = α·current + (1-α)·previous EMA
check_circle Responsive to changes
check_circle Less memory usage

functions Savitzky-Golay Filter

Fits polynomial to local data points
check_circle Preserves higher moments
check_circle Good for peak preservation
check_circle Adjustable polynomial degree

blur_on Gaussian Filter

Uses Gaussian function for weighted averaging
check_circle Natural smoothing effect
check_circle Adjustable sigma parameter
check_circle Good for noise reduction

filter_center_focus Median Filter

Replaces value with median of neighboring points
check_circle Effective against outliers
check_circle Preserves edges
check_circle Non-linear filtering

trending_up Double Exponential Smoothing

Extends exponential smoothing with trend component
Level: L = α·Y + (1-α)·(L+T)
Trend: T = β·(L-Lₚᵣₑᵥ) + (1-β)·T
check_circle Captures trends
check_circle Good for forecasting

euro One Euro Filter

Adaptive filter for real-time noisy signals
check_circle Adjusts to signal speed
check_circle Reduces jitter without lag
check_circle Good for interactive systems

bezier_curve Bezier Curve Smoothing
